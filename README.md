# Көчмөн Өрнөк

`Көчмөн Өрнөк` - HTML, CSS жана JavaScript менен жасалган браузердик мини-оюн.
Оюн түз эле браузерде иштейт жана орнотууну талап кылбайт.

## Оюндун максаты
Берилген фигуралардан бир түстөгү 3 же андан көп клетканы айкалыштырып, аларды тазалап, мүмкүн болушунча көп упай топтоо.

## Колдонулган технологиялар
- `HTML` - барактын түзүмү
- `CSS` - көрүнүшү, анимация жана этно-стиль
- `JavaScript` - оюндун логикасы жана эрежелери

## Башкаруу
- Чычкандын сол баскычы - блокту тандоо
- Чычкан менен сүйрөө - блокту торчого жайгаштыруу

## Долбоордун түзүмү
- `index.html` - негизги бет
- `c.css` - стили жана визуалдык тема
- `j.js` - оюн логикасы

## Иштетүү
1. Репозиторийди жүктөп алыңыз же клон кылыңыз.
2. `index.html` файлын браузерде ачыңыз.

## Талаптар
- Заманбап браузер: Chrome, Firefox, Edge же Safari
